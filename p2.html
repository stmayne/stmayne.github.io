Super Hero IDB

Welcome to the project page for superhero IDB

Superhero IDB is a project I created with a group in the Spring of 2015 for our software enginering course at the University of Texas at Austin.
The project was created to have a website that would allow for quick and easy viewing of pages that showed information about superheros, comic books, and marvel based movies.
To create the site we used technologies include python, HTML, CSS, Twitter Bootstrap and others.
